Detailed Description

Stores uniform values for a particular shader program.

Stores the uniform values for a particular shader program by maintaining a mapping from uniforms' names to their values. This class provides a way of storing all the uniform values of a shader_program. By convention any name provided will be converted to the uniform equivalent which has a u prefixed to the beginning and the first letter of the original name capitalized( "model" becomes "uModel" ).
