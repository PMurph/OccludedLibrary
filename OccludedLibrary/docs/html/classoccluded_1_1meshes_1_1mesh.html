Detailed Description

An abstract class that provides an interface for all mesh classes.

An abstract class that provides an interface for all mesh classes. Since meshes are one of the basic objects that will be used in the occluded library, a common interface must be created for all meshes so that any graphics API can be used without changing the way in which the library is used by the end user.

Adds a single face to the mesh.

Parameters
faceIndicesA reference to a vector of unsigned ints containing the indices of the vertices that make up the face to be added.
Returns
A unsigned int representing the index of the face that was added.

Adds a face to the mesh that is made up of the vertices specified by the indices in the faceIndices vector. An exception is thrown if the faceIndices vector does not contain the correct number of vertices or if it contains a index that does not correspond to a vertex in the the mesh.

Parameters

Returns an unsigned int representing the number of vertices needed to add a new face.

Gets the number of vertices needed for the next face to be added to the mesh after the face specified by numFaces parameter. This should be called before adding a face to the mesh so that the chance of an exception being thrown by the call is minimized.
