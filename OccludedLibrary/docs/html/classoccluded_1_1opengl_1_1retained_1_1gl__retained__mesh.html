Detailed Description

A mesh implementation for OpenGL retained mode.

A mesh implementation for OpenGL retained mode. This contains a gl_attribute_buffer and a vector of unsigned int representing the faces of the of the mesh. This is class contains everything needed to draw a single object in object coordinates. The purpose of this class is to be contained within a model class and allow the model class to render the mesh with a single call. /see { occluded::opengl::retained::gl_attribute_buffer }

This constructor is to be used if the mesh is to be built up from scratch. An exception will be thrown in the attribute map is still being defined or if the shader program has not been linked.
